PROACT (Feb 2019 – Oct 2021): "Planetary RObots Deployed for Assembly and Construction Tasks".

Focused on collaboration in planetary robotics, this project focused on extending OG2 autonomy capabilities to a multi-agent system. Using this approach, a fleet of robots could autonomously execute a representative set of operations to build an ISRU plant. H2020 SRC PERASPERA OG11.
